# NEWSPAD

A News Application made up of JavaScript library ReactJS.

Through this Application you can read Latest News in various topic like :-

- #### GENERAL

- #### ENTERTAINMENT

- #### TECHNOLOGY

- #### BUSINESS

- #### HEALTH

- #### SCIENCE

- #### SPORTS

- Clean and simple Application.
